    #Gestion de Parking
Ce projet vise à fournir une solution de gestion de parking à la fois pour les utilisateurs mobiles et web. Il utilise Flutter pour le développement des interfaces utilisateur mobiles et web, et une REST API pour la communication entre le frontend et le backend.
-Technologies utilisées
Flutter : Framework de développement d'applications multiplateformes pour les interfaces utilisateur mobiles et web.
Dart : Langage de programmation utilisé avec Flutter.
REST API : Pour la communication entre le frontend et le backend.
-Configuration requise
Flutter SDK 
Un éditeur de code (intellij, Android Studio)
